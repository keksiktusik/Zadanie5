Funkcja obliczająca wartość całki w zadanym przedziale metodą prostokątów.

Funkcja wykorzystuje metodę prostokątów, aby przybliżyć wartość całki funkcji ( f(x) ) na zadanym przedziale [start, end]. Całkowity przedział całkowania jest dzielony na steps równych części, a wynik z każdej części jest sumowany.

Parameters
startPoczątek przedziału całkowania.
endKoniec przedziału całkowania.
stepsLiczba kroków podziału w przedziale. Większa liczba kroków zwiększa dokładność.
stepSizeRozmiar jednego kroku (delta x). Określa szerokość prostokąta.
resultZmienna wynikowa, do której zapisany zostanie wynik obliczeń.

Wyjaśnienie działania:

  • Każdy krok reprezentuje prostokąt, którego szerokość to stepSize, a wysokość to wartość funkcji ( f(x) ) w środku tego prostokąta.
  • Wynik dla danego prostokąta jest obliczany jako ( f(x) \times \text{stepSize} ), a wszystkie wyniki są sumowane w zmiennej result.

< Dodanie pola prostokąta do sumy.

< Zapisanie wyniku całkowania w przedziale do zmiennej result.

double f ( double x)

Funkcja obliczająca wartość funkcji ( f(x) = \frac{4}{1 + x^2} ).

Funkcja ta jest kluczowa w obliczaniu liczby PI metodą całkowania numerycznego. Reprezentuje funkcję, której całka w granicach [0, 1] daje wartość liczby PI.

Parameters
xPunkt, w którym funkcja ma zostać obliczona.
Returns
Wartość funkcji ( f(x) ) w punkcie x.

Wyjaśnienie:

Funkcja ( f(x) = \frac{4}{1 + x^2} ) pochodzi z przekształcenia równania koła ( x^2 + y^2 = 1 ). Wynik integracji tej funkcji w zakresie od 0 do 1 równa się liczbie PI.

◆ main()

int main ( )

Funkcja główna programu.

Funkcja sterująca, która wykonuje obliczenia liczby PI metodą całkowania numerycznego dla różnych konfiguracji liczby wątków i kroków. Program obsługuje wielowątkowość, dzieląc obliczenia na równoległe fragmenty, aby przyspieszyć działanie. Wyniki są zapisywane w pliku CSV, co pozwala na analizę wpływu liczby wątków i kroków na wydajność.

Returns
Zwraca 0, jeśli program zakończył się poprawnie.

Wyjaśnienie:

  • Funkcja iteruje przez różne liczby kroków (dokładności obliczeń) oraz różne liczby wątków (poziomy równoległości).
  • Dla każdej kombinacji liczby kroków i wątków, funkcja:
    • Dzieli zakres obliczeń między wątki.
    • Uruchamia wątki do równoległego obliczania wartości całki.
    • Sumuje wyniki z poszczególnych wątków.
    • Zapisuje wyniki do pliku CSV.
